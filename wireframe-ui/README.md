# Wireframe UI

A self-contained wireframe UI for the chat application.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `components/` - All React components
- `app/` - Next.js app directory with layout and styles
- `lib/` - Utility functions
- `hooks/` - React hooks
- `page.tsx` - Main wireframe page component

## Features

- Self-contained UI wireframe
- All dependencies included
- No external API calls
- Fake message sending functionality
